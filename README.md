# Sistema de Gerenciamento de Portaria

## 📌 Sobre o projeto
Sistema web desenvolvido para controle de acesso em condomínios, permitindo o gerenciamento de moradores, visitantes e prestadores de serviço com foco em segurança e rastreabilidade.

## 🚀 Funcionalidades
- Autenticação de usuários
- Cadastro e gerenciamento de moradores, visitantes e funcionários
- Registro de entrada e saída com data e hora
- Captura de imagem via webcam para identificação
- Consulta de dados por CPF
- Exclusão com validação de integridade (vínculos entre registros)
- Geração de relatórios de acesso

## 🧠 Diferenciais
- Controle de integridade relacional (impede exclusões inconsistentes)
- Identificação visual com captura de imagem
- Sistema pensado para uso real em portarias

## 🛠 Tecnologias
- PHP (back-end)
- MySQL (banco de dados)
- HTML, CSS, JavaScript (front-end)
